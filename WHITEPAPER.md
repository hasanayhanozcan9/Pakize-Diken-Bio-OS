## 2. Core Architectural Protocols

### 2.1 PD-MIP (Molecular Internet Protocol)
The transport layer for molecular packets. Utilizing the Stokes-Einstein equation and Fick’s laws of diffusion, PD-MIP manages stochastic data transmission within biological fluids.

### 2.2 BDE (Bio-Dynamic Encryption) - V4
Unlike static cryptographic keys, BDE-V4 generates temporal keys derived from the host's real-time physiological states (heart rate variability, glucose fluctuations, and quantum coherence). The host is the key.

### 2.3 LLG (Liquid Logic Gates) & LNN (Liquid Neural Networks)
Replacing silicon transistors with chemical reaction networks (CRNs). LNN allows for time-continuous adaptation, governed by:
$$\frac{dx(t)}{dt} = -[A + B \cdot f(x(t))]x(t) + I(t)$$
This ensures the system adapts its logic as the biological environment evolves.

### 2.4 SNN (Spiking Neuromorphic Core)
An event-driven architecture that ensures zero-energy consumption during homeostasis. The OS "spikes" into action only when biological stimulus crosses the action potential threshold (-55mV).
